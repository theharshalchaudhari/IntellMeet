# IntellMeet Landing Page

Official marketing website for IntellMeet.

Built with Astro, React, Tailwind CSS, and modern web performance practices.

The landing page introduces IntellMeet, showcases platform capabilities, communicates the product vision, and guides users into the IntellMeet ecosystem.

---

# About

IntellMeet is an AI-powered meeting intelligence platform designed for teams, startups, colleges, and organizations.

The platform transforms meetings into searchable knowledge systems through:

* Real-time collaboration
* AI-generated summaries
* Action item extraction
* Meeting memory
* Workspace collaboration
* Intelligent search

This repository contains the public-facing website for the IntellMeet platform.

---

# Development Setup

## Requirements

* Node.js 18+
* pnpm 9+

```bash
pnpm install
```

## Run Development Server

```bash
pnpm dev
```

## Build

```bash
pnpm build
```

## Preview Production Build

```bash
pnpm preview
```
